### Installation

1. **Clone and setup**
   ```bash
   git clone https://github.com/company/secure-flow-automaton.git
   cd secure-flow-automaton
   npm install
   ```

2. **Configuration**
   ```bash
   cp .env.example .env
   # Edit .env with your settings
   ```

3. **Database setup**
   ```bash
   npm run db:migrate
   npm run db:seed
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```
